3분 설치 가이드

01

환경 준비

Windows 10/11 환경에서 Python 32-bit 버전을 설치하세요.

02

모듈 설치

pip install -r requirements.txt 명령어로 필수 라이브러리를 설치합니다.

03

시스템 실행

키움증권 로그인 후 python main.py를 실행하면 자동매매가 시작됩니다.

수익 극대화 노하우

1만원으로 10만원을 만드는 1,000% 성과는 '기계적인 원칙'에서 시작됩니다.

  • 오전 9시 ~ 9시 30분의 폭발적인 변동성 집중 공략
  • 자가 진화 알고리즘을 통한 최적의 K-값 자동 산출
  • -1.5% 손절 원칙을 통한 자산 보호
