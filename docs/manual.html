사용자 매뉴얼

시스템 가동부터 결과 분석까지, 모든 과정을 세밀하게 가이드합니다.

  • config.yaml 설정
  • 프로그램 가동
  • 대시보드 활용
  • 자주 묻는 질문

1. config.yaml 완벽 마스터

시스템의 두뇌인 설정 파일을 본인의 스타일에 맞게 튜닝하세요.

항목설명권장값
max_budget자동매매에 사용할 총 예산계좌 잔고의 50%
stop_loss손절매 기준 (비율)0.015 (-1.5%)
k_value돌파 민감도 (낮을수록 공격적)0.4 ~ 0.6
gemini_api_keyAI 뉴스 분석용 API 키필수

2. 프로그램 가동 프로세스

  1. KOA Studio 실행: 키움증권 서버 상태를 먼저 확인하세요.
  2. main.py 실행: python main.py를 관리자 권한으로 실행합니다.
  3. 로그인 확인: 화면 우측 하단 트레이 아이콘에 'OpenAPI'가 정상 활성화되었는지 확인하세요.
  4. 텔레그램 확인: "자동매매가 시작되었습니다" 메시지가 오면 정상입니다.

3. 실시간 대시보드 활용

streamlit run monitor/dashboard.py 명령어로 웹 대시보드를 켭니다.

  • Equity Curve: 자산의 우상향 여부를 체크하세요.
  • Heatmap: 오늘 어떤 종목이 효자 종목인지 시각적으로 확인하세요.
  • AI Review: 장 종료 후 AI가 분석한 오늘의 매매 복기를 정독하세요.

4. 자주 묻는 질문 (FAQ)

Q: 주문이 나가지 않아요.

A: 1. 예수금이 부족하지 않은지 확인하세요. 2. 모의투자 계좌인 경우 키움증권 홈페이지에서 '모의투자 신청'이 되어 있는지 확인하세요.

Q: 64비트 파이썬을 사용하면 안 되나요?

A: 불가능합니다. 키움증권 API 라이브러리가 32비트 전용으로 설계되었기 때문입니다.
