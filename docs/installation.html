상세 설치 가이드

Stock Quant Trader를 당신의 컴퓨터에 이식하는 가장 확실한 방법입니다.

Step 1: 시스템 요구사항 확인

  • 운영체제: Ubuntu/Linux, Windows, macOS 모두 지원 (REST API 방식)
  • 하드웨어: RAM 8GB 이상 권장 (안정적인 데이터 처리를 위함)
  • 보안: 관리자 권한 실행 권장

Step 2: Python 설치

한국투자증권 KIS API는 64비트 및 모든 환경을 완벽 지원합니다.

  1. Python 공식 사이트 방문
  2. 해당 OS에 맞는 Python 다운로드 및 실행
  3. 설치 시 Add Python to PATH 옵션 반드시 체크

Step 3: 한국투자증권 OpenAPI 서비스 신청

  1. 한국투자증권 KIS Developers 로그인
  2. App Key 및 App Secret 발급
  3. 모의투자 계좌 개설 (권장)

Step 4: 프로젝트 환경 구축

git clone https://github.com/leemgs/stock-quant-trader.git
cd stock-quant-trader
pip install -r requirements.txt
