상세 설치 가이드

K-Quant Pro를 당신의 컴퓨터에 이식하는 가장 확실한 방법입니다.

Step 1: 시스템 요구사항 확인

  • 운영체제: Windows 10 또는 11 필수 (32-bit API 호환성 때문)
  • 하드웨어: RAM 8GB 이상 권장 (안정적인 데이터 처리를 위함)
  • 보안: 관리자 권한 실행 권장

Step 2: Python 32-bit 설치

키움증권 API는 32비트 전용입니다. 반드시 32비트 버전을 설치해야 합니다.

  1. Python 공식 사이트 방문
  2. Windows installer (32-bit) 다운로드 및 실행
  3. 설치 시 Add Python to PATH 옵션 반드시 체크

Step 3: 키움 OpenAPI+ 설치 및 서비스 신청

  1. 키움증권 홈페이지 로그인
  2. [전체메뉴] > [트레이딩] > [오픈API] > [서비스 신청]
  3. 모듈 다운로드 및 설치
  4. KOA Studio 설치 (데이터 구조 확인용으로 매우 유용)

Step 4: 프로젝트 환경 구축

git clone https://github.com/leemgs/kiwoom-quant-trader.git
cd kiwoom-quant-trader
pip install -r requirements.txt
